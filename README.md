# Веб-приложение поликлиники

Курсовая работа. Веб-приложение для управления записью пациентов в поликлинику.

Стек: Python, Flask, PostgreSQL, Docker.

**Демо:** https://web-course-work.onrender.com

**Репозиторий:** https://github.com/ZZAY-GIT/web-course-work

## Что умеет

- Регистрация и авторизация пациентов
- Запись на приём к врачу с выбором услуги и времени
- Личный кабинет пациента с историей посещений и медицинскими документами
- Кабинет врача с расписанием и возможностью оформить результаты приёма
- Панель администратора для управления врачами, услугами и расписанием

## Запуск через Docker (рекомендуется)

Требования: Docker и Docker Compose.

```bash
docker compose up --build
```

После запуска приложение будет доступно по адресу: http://localhost:5000

База данных заполняется тестовыми данными автоматически при первом запуске.

## Запуск без Docker

Требования: Python 3.11+, PostgreSQL.

1. Установить зависимости:

```bash
cd clinic_app
pip install -r requirements.txt
```

2. Задать переменную окружения с адресом базы данных:

```bash
set DATABASE_URL=postgresql://user:password@localhost:5432/clinic_db
```

3. Инициализировать базу и запустить приложение:

```bash
python seed.py
python app.py
```

Приложение будет доступно по адресу: http://localhost:5000
